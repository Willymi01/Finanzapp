# Meine Finanzzentrale V6.0

GitHub-Pages-fertige Progressive Web App mit:

- Finanzplan
- monatlichem Sparplan
- Vermögens- und Finanzierungsprognose
- Datums- und Zwischenstandslogik
- Installation auf PC und Smartphone
- Offline-Cache
- optionaler Anmeldung und Cloud-Synchronisierung über ein eigenes Supabase-Projekt

Die vollständige Einrichtung steht in `EINRICHTUNG.md`.

Wichtig: `config.js` enthält zunächst keine Zugangsdaten. Ohne Supabase-Konfiguration funktioniert die App weiterhin lokal.
